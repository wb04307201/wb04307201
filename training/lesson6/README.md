## 为什么需要多智能体？

目前 AI Agent 的应用正从**单智能体**解决单一问题转向**多智能体**解决复杂场景。`单智能体 vs 多智能体` 就像 `瑞士军刀 vs 工具箱`：

- 单智能体 = 一把瑞士军刀，什么都能干但都不精
- 多智能体 = 一整套专业工具箱，每个工具专注一个领域
