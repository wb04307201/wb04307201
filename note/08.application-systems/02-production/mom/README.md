# MOM · 制造运营管理系统

> 一份按业务场景梳理的 MOM 速查手册：从车间执行到生产管理的"中枢神经系统"。

---

## 一、一句话定位

**MOM（Manufacturing Operations Management，制造运营管理系统）**：在 MES 之上，串联生产、库存、质量、维护等所有工厂运营环节的"中枢系统"，目标是"用数据驱动工厂高效运营"。

---

## 二、核心功能（6 大模块）

| 模块 | 功能 | 价值 |
|------|------|------|
| **生产运营** | 生产计划、工序调度、车间执行 | 提升产能利用率 10-30% |
| **质量管理** | 来料/过程/成品检验、SPC 统计 | 降低不良率 |
| **库存管理** | 原材料/半成品/成品库存 | 减少呆滞库存 |
| **设备维护** | 预防性维护、OEE 分析 | 减少停机时间 |
| **追溯管理** | 正反向追溯、批次追踪 | 召回时间从天→小时 |
| **能源管理** | 水电气消耗、KPI 监控 | 降低能耗 5-15% |

---

## 三、MES vs MOM 区别

| 维度 | MES | MOM |
|------|-----|-----|
| 范围 | 单一车间执行 | 全工厂运营（含质量/库存/设备） |
| 核心 | 实时执行（工单/工序）| 运营管理（计划/分析/决策）|
| 用户 | 车间主任 / 班组长 | 工厂厂长 / 生产总监 |

**关系**：MOM = MES + QMS（质量）+ EAM（设备）+ WMS（库存）的整合视图。

---

| 角色 | 主要诉求 |
|------|---------|

## 八、实施关键点

1. **数据采集**：先解决"数据从哪来"（PLC / SCADA / 手工录入）
2. **主数据治理**：BOM、工序、设备编码必须先标准化
3. **小步快走**：先上工单执行（3 个月）+ 后续模块（质量/设备）
4. **变更管理**：一线员工抵触最大，要配激励

## 九、MOM 深度集成与扩展

MOM 作为工厂的"中枢神经系统"，其价值取决于与上下游系统的集成深度和数据分析能力。

### 9.1 MOM 的 ISA-95 分层模型

```text
Level 4 ─── ERP（业务计划：订单/成本/库存）
              ↕
Level 3 ─── MOM（运营管理：调度/质量/设备/追溯）
              ↕
Level 2 ─── SCADA（监控：实时数据采集 + HMI）
              ↕
Level 1 ─── PLC/传感器（控制：执行器 + 信号采集）
              ↕
Level 0 ─── 物理过程（生产过程）
```

### 9.2 MOM 与 MES/APS/SCADA 的协作模式

| 协作关系 | 数据流 | 协作频率 | 关键接口 |
|---------|--------|---------|---------|
| **MOM ↔ MES** | MOM 下发调度指令 → MES 回传执行状态 | 实时 | 工单状态、工序进度、异常上报 |
| **MOM ↔ APS** | APS 下发排程 → MOM 分配资源执行 | 日/班次 | 排程计划、产能反馈 |
| **MOM ↔ SCADA** | SCADA 采集设备数据 → MOM 分析 OEE | 秒级 | 设备状态、工艺参数、报警信号 |
| **MOM ↔ QMS** | MOM 触发质检 → QMS 回传结果 | 批次级 | 检验任务、不良品处理 |

### 9.3 MOM 数字化转型路径

| 阶段 | 目标 | 关键动作 |
|------|------|---------|
| **L1 数字化** | 数据能采上来 | SCADA 全覆盖 + PLC 联网 |
| **L2 信息化** | 数据能看得见 | MES + MOM 看板上线 |
| **L3 智能化** | 数据能分析 | SPC + OEE 分析 + 追溯 |
| **L4 自适应** | 系统能决策 | AI 排产 + 预测性维护 + 数字孪生 |

### 9.4 关键实施考量

| 考量点 | 建议 |
|--------|------|
| **SCADA 选型** | 优先选支持 OPC-UA 的 SCADA（与 MOM 标准化对接）|
| **数据频率** | 设备状态 ≥ 1 次/秒，质量数据按批次，能耗按 15 分钟 |
| **边缘计算** | 高频数据在边缘节点预处理，只上传聚合结果到 MOM |
| **多工厂部署** | 每个工厂独立 MOM 实例 + 集团级数据湖汇总 |
