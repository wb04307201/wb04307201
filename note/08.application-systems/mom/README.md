## 一、一句话定位

**MOM（Manufacturing Operations Management，制造运营管理系统）**：在 MES 之上，串联生产、库存、质量、维护等所有工厂运营环节的"中枢系统"，目标是"用数据驱动工厂高效运营"。

---

## 二、核心功能（6 大模块）

| 模块 | 功能 | 价值 |
|------|------|------|
| **生产运营** | 生产计划、工序调度、车间执行 | 提升产能利用率 10-30% |
| **质量管理** | 来料/过程/成品检验、SPC 统计 | 降低不良率 |
| **库存管理** | 原材料/半成品/成品库存 | 减少呆滞库存 |
| **设备维护** | 预防性维护、OEE 分析 | 减少停机时间 |
| **追溯管理** | 正反向追溯、批次追踪 | 召回时间从天→小时 |
| **能源管理** | 水电气消耗、KPI 监控 | 降低能耗 5-15% |

---

## 三、MES vs MOM 区别

| 维度 | MES | MOM |
|------|-----|-----|
| 范围 | 单一车间执行 | 全工厂运营（含质量/库存/设备） |
| 核心 | 实时执行（工单/工序）| 运营管理（计划/分析/决策）|
| 用户 | 车间主任 / 班组长 | 工厂厂长 / 生产总监 |

**关系**：MOM = MES + QMS（质量）+ EAM（设备）+ WMS（库存）的整合视图。

## 八、实施关键点

1. **数据采集**：先解决"数据从哪来"（PLC / SCADA / 手工录入）
2. **主数据治理**：BOM、工序、设备编码必须先标准化
3. **小步快走**：先上工单执行（3 个月）+ 后续模块（质量/设备）
4. **变更管理**：一线员工抵触最大，要配激励
