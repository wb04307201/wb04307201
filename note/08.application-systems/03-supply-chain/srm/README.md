# SRM · 供应商关系管理系统

> 一份按业务场景梳理的 SRM 速查手册：从"管供应商"到"经营供应商生态"。

---

## 一、一句话定位

**SRM（Supplier Relationship Management，供应商关系管理系统）**：把"供应商"从外部对手变成"合作伙伴"，通过全生命周期管理（寻源 → 准入 → 合作 → 评估 → 退出），最大化供应商价值。

---

## 二、核心功能（6 大模块）

| 模块 | 功能 | 价值 |
|------|------|------|
| **供应商寻源** | 公开招标、邀请招标、询比价 | 引入新供应商 |
| **供应商准入** | 资质审核、背景调查 | 降低风险 |
| **合同管理** | 合同模板、电子签章、到期提醒 | 规避法律风险 |
| **订单协同** | PO 下达、交期确认、送货预约 | 减少沟通成本 |
| **对账结算** | 3 方对账（采购方/供应商/ERP）、发票管理 | 减少财务纠纷 |
| **绩效评估** | QCDS（质量/成本/交付/服务）打分 | 优胜劣汰 |

---

## 三、SRM vs SCM vs ERP 关系

| 系统 | 关注对象 | 关注内容 |
|------|---------|---------|
| **SRM** | 供应商（外部）| 准入 / 评估 / 协同 |
| **SCM** | 整条链（端到端）| 链优化 / 网络协同 |
| **ERP** | 内部资源 | 订单 / 库存 / 财务 |

**关系**：SRM 是 SCM 的"前端模块"，SCM 包含 SRM。

---

## 四、典型用户与角色

| 角色 | 主要诉求 |
|------|---------|
| 采购总监 | 供应商生态优化、降本 |
| 采购经理 | 日常订单、对账、绩效 |
| 质量经理 | 来料质量协同 |
| 供应商 | PO 接单、送货预约、对账 |

---

## 五、与上下游系统的集成

```text
上游（供应商侧）         SRM 本体           下游（采购方侧）
─────────────────   ─────────────   ──────────────────
供应商 ERP            →             ←   采购方 ERP（PO）
供应商 Portal        →             ←   采购方 WMS（收货）
CA 认证/电子签章     →   订单/对账    ←   财务系统（结算）
质检系统（来料检验）   →             ←   BI（绩效分析）
```

## 八、实施关键点

1. **供应商主数据治理**：统一编码、统一分类
2. **流程梳理**：从"线下微信群沟通"到"线上流程化"
3. **绩效模型**：QCDS 4 维度 + 业务相关维度（合规/ESG）
4. **Portal 推广**：让供应商主动用起来（送培训/积分激励）
5. **数据反哺**：供应商绩效反哺采购决策（招标/份额）

## 九、SRM 深度应用场景

### 9.1 供应商协同 Portal

SRM 的核心价值不仅在"管理"，更在"协同"——让供应商从被动接单变为主动参与：

| 协同场景 | 功能 | 价值 |
|---------|------|------|
| **需求预发布** | 采购方发布未来 3 个月预测需求 | 供应商提前备料，缩短交期 20-30% |
| **在线竞标** | 反向拍卖（Reverse Auction） | 采购成本降低 8-15% |
| **自助送货** | 供应商自助预约送货时间 | 减少仓库等待时间 50% |
| **在线对账** | 三方对账（采购方 / 供应商 / 财务）| 对账周期从 15 天 → 3 天 |

### 9.2 SRM 与 ERP 的集成深度

SRM 与 ERP 的集成是"采购到付款（Procure-to-Pay）"流程的关键链路：

```text
SRM 侧                    ERP 侧
────────                 ────────
供应商主数据 ──────→ 供应商主数据同步
采购申请（PR）←────── 需求部门提报
采购订单（PO）──────→ PO 自动生成
收货确认     ←────── WMS 回传
发票校验     ←────── 财务应付
付款状态     ←────── 财务已付
```

### 9.3 选型决策树

| 判断条件 | 推荐方案 |
|---------|---------|
| 年采购额 > 10 亿 + 供应商 > 500 家 | SAP Ariba / Oracle SRM（全功能 + 全球化）|
| 已有 SAP/Oracle ERP | 选同厂商 SRM（集成最顺畅）|
| 年采购额 1-10 亿 + 国产 ERP | 用友友云采 / 金蝶（与国产 ERP 生态匹配）|
| 中小企业 + 轻量需求 | 甄云 / 携客互联 SaaS（3 个月上线）|
