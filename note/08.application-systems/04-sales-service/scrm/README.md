# SCRM · 社交化客户关系管理

> 一份按业务场景梳理的 SCRM 速查手册：从"客户管理"到"客户经营"。

---

## 一、一句话定位

**SCRM（Social CRM，社交化客户关系管理）**：把 CRM 从"内部管理工具"升级为"客户全生命周期运营平台"，核心是基于微信/抖音/小红书等社交渠道，把客户变成"粉丝"，把粉丝变成"复购用户"。

---

## 二、SCRM vs CRM 区别

| 维度 | 传统 CRM | SCRM |
|------|---------|------|
| 核心 | 客户档案管理 | 客户全旅程运营 |
| 数据来源 | 销售录入 | 社交行为 + 销售数据 |
| 客户关系 | 单向（公司 → 客户）| 双向（社交互动）|
| 触达渠道 | 邮件 / 短信 | 微信 / 抖音 / 小红书 / 社群 |
| 核心指标 | 客户数量 | 客户 LTV / 复购率 / 传播力 |

---

## 三、核心功能（6 大模块）

| 模块 | 功能 | 价值 |
|------|------|------|
| **客户画像** | 整合社交行为 + 交易数据 | 360° 客户视图 |
| **内容营销** | 朋友圈/社群内容推送 | 提升触达率 |
| **私域运营** | 企业微信群运营 | 提升触达与转化（行业差异大，无统一公开口径） |
| **会员体系** | 积分/等级/权益 | 提升复购 |
| **裂变拉新** | 拼团/分销/砍价 | 降低获客成本 |
| **数据看板** | 行为分析 + ROI 看板 | 实时决策 |

---

## 四、与微信生态的集成（国内核心场景）

```text
微信公众号 ─→ SCRM 内容推送
微信小程序 ─→ SCRM 电商交易
企业微信 ──→ SCRM 客户群运营
视频号 ────→ SCRM 直播带货
朋友圈 ────→ SCRM 朋友圈营销
微信支付 ──→ SCRM 交易数据回流
```

---

## 五、典型行业应用

| 行业 | 重点 |
|------|------|
| **美妆/服饰** | KOL 种草 + 社群运营 + 直播带货 |
| **母婴/教育** | 知识付费 + 社群分销 |
| **餐饮/零售** | 会员体系 + 优惠券 + 复购 |
| **B2B** | 客户成功 + 续费管理 |
| **医美/健康** | 私域 + 信任构建 |

---

## 六、主流厂商/产品对比

| 类型 | 代表产品 | 适用场景 |
|------|---------|---------|
| **综合型** | 微盟 / 有赞 / 销售易 | 通用 |
| **垂直型** | 群脉 / 咚咚 / 微盛 | 私域运营 |
| **微信生态原生** | 企微 SCRM / 微伴 | 企业微信深度 |
| **SaaS 新势力** | 小鹅通 / 知识星球 | 知识付费 |
| **国际型** | Hubspot / Salesforce Marketing Cloud | 海外 |

---

## 七、选型建议

| 场景 | 推荐 |
|------|------|
| 中小零售 / 电商 | 有赞 / 微盟 |
| 大型零售集团 | 微盟 + 自研 |
| 知识付费 / 教育 | 小鹅通 / 知识星球 |
| 企业微信为主 | 微伴 / 企微 SCRM |
| 多渠道整合 | 销售易 / Salesforce |
| B2B 企业 | 销售易 / 神州云动 |

---

## 八、实施关键点

1. **客户数据整合**：打通 CRM + 电商 + 社交数据
2. **内容运营**：内容是 SCRM 的核心，不能只上系统
3. **企业微信对接**：企业微信是 SCRM 的最佳载体
4. **KPI 设计**：从"客户数"转向"客户 LTV / 复购率"
5. **合规风险**：用户授权、数据合规要重点关注
