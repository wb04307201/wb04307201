# Gitea

Gitea 是一款**轻量级、开源、自托管**的 Git 服务软件，旨在提供类似 GitHub/GitLab 的代码托管、协作开发和 DevOps 功能，但对服务器资源要求极低，易于部署和维护。它由 Go 语言编写，是 **Gogs 项目的社区分支**（因原团队协作分歧诞生），目前已成为全球最受欢迎的自托管 Git 服务之一。

---

### **核心特点**
1. **极致轻量 & 低资源消耗**
    - 单二进制文件部署，无需复杂依赖。
    - 最低配置：**512MB 内存 + 1核 CPU** 即可流畅运行（适合树莓派、老旧服务器或低成本云主机）。
    - 启动速度快，内存占用通常低于 100MB。

2. **功能全面**
    - **基础功能**：代码仓库管理、分支/标签、Pull Request、Issue 跟踪、Wiki 文档、代码审查。
    - **协作工具**：团队/组织管理、权限控制（读/写/管理）、审计日志、邮件通知。
    - **DevOps 集成**：  
      ✅ **Gitea Actions**（类 GitHub Actions，2023 年正式 GA）：支持 CI/CD 流水线。  
      ✅ Webhook & API（兼容 GitHub API）  
      ✅ 包注册表（Container Registry、Maven、npm、PyPI 等）
    - **扩展生态**：支持 OAuth2、LDAP/AD 认证、SAML SSO、两步验证、第三方登录（微信/钉钉等需插件）。

3. **跨平台 & 部署简单**
    - 支持 **Linux/Windows/macOS/ARM**（树莓派友好）。
    - 一键部署方式：
      ```bash
      # Docker 部署（推荐）
      docker run -d --name gitea -p 3000:3000 -v ./gitea:/data gitea/gitea:latest
      ```
    - 也提供二进制包、Helm Chart（K8s）、系统服务脚本等。

4. **高性能 & 稳定性**
    - 基于 Go 的高并发模型，万级用户实测稳定。
    - SQLite/MySQL/PostgreSQL/ MSSQL 多数据库支持（默认 SQLite，适合小团队快速启动）。

5. **100% 开源 & 社区驱动**
    - **Apache 2.0 许可证**：可免费用于商业场景。
    - 活跃的全球社区（GitHub 58k+ Stars），中文文档完善，问题响应迅速。
    - 无付费墙/企业版功能阉割，所有功能开源。

---

### **适用场景**
- **中小企业/创业团队**：低成本搭建私有代码托管平台（替代 GitLab 高资源消耗）。
- **个人开发者**：在低配 VPS 或 NAS 上托管个人项目。
- **教育/科研机构**：内网隔离环境下的代码协作。
- **IoT/边缘设备**：在资源受限设备（如树莓派）上运行。
- **对数据隐私敏感的场景**：完全掌控代码和数据，避免云服务商锁定。

### **与竞品对比**
| 特性                | Gitea                     | GitLab CE                | Gogs                     |
|---------------------|---------------------------|--------------------------|--------------------------|
| **资源占用**        | ⭐ 极低 (512MB 内存可用)   | ❌ 高 (建议 4GB+ 内存)    | ⭐ 低 (但功能较少)        |
| **功能丰富度**      | ✅ 全面 (Actions/包注册表) | ✅ 最全面 (含监控/安全)   | ❌ 基础功能               |
| **部署复杂度**      | ✅ 极简 (单文件)           | ❌ 复杂 (依赖多组件)       | ✅ 极简                  |
| **社区活跃度**      | ✅ 高 (持续快速迭代)       | ✅ 高                    | ⚠️ 低 (更新缓慢)         |
| **学习曲线**        | ⭐ 低 (界面类似 GitHub)    | ⚠️ 中                    | ⭐ 低                   |

> 💡 **简单说**：
> - 要 **轻量+易用+够用** → 选 **Gitea**
> - 需要 **企业级安全/监控/项目管理** → 选 GitLab
> - 仅需 **最简 Git 服务** 且无扩展需求 → 考虑 Gogs

### **快速开始**
1. **访问官网**：[https://gitea.io](https://gitea.io)（中文界面完善）
2. **5 分钟部署**：
    - Docker 用户：`docker run -d --name gitea -p 3000:3000 gitea/gitea:latest`
    - 访问 `http://服务器IP:3000` 按向导初始化。
3. **中文文档**：[https://docs.gitea.io/zh-cn/](https://docs.gitea.io/zh-cn/)
