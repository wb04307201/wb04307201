# 算法概述

> 算法是**解决问题的清晰指令序列**——一系列有限步骤的精确描述，将输入转换为输出。

---
---

## 一、核心特征

| 特征 | 说明 |
|------|------|
| **可行性** | 每一步可分解为基本可执行操作 |
| **确定性** | 每一步明确定义，无歧义 |
| **有穷性** | 在有限步骤内终止 |
| **输入/输出** | 0+ 个输入 → 1+ 个输出 |

> "算法 + 数据结构 = 程序" — 尼克劳斯·维尔特

---

## 二、分类体系

### 按设计方法

| 类别 | 核心思想 | 典型算法 | 适用场景 |
|------|---------|---------|---------|
| **贪心算法** | 每步选局部最优 | Dijkstra、Kruskal、Huffman | 最短路、最小生成树、编码压缩 |
| **动态规划** | 子问题 + 存储中间结果 | 背包问题、LCS、编辑距离 | 最优化、计数 |
| **分治法** | 分而治之、合并结果 | 快排、归并、FFT | 大规模数据处理 |
| **回溯法** | 枚举 + 剪枝 | N 皇后、数独、全排列 | 组合优化 |
| **随机化** | 引入随机性 | 蒙特卡罗、快速排序随机化 | 近似解、平均性能优化 |
| **分支限界** | BFS + 剪枝 | TSP、0-1 背包 | 整数规划 |

### 按应用领域

- **基础算法** — 排序/查找/递归与迭代
- **数据结构相关** — 堆/栈/队列/树/图
- **图论算法** — BFS/DFS/最短路径/最小生成树/网络流/拓扑排序
- **字符串算法** — KMP、Rabin-Karp、Trie、后缀数组
- 🆕 **[字符串算法深度专题](string-algorithms/README.md)** —— Trie（字典树）/ KMP / AC 自动机（Aho-Corasick）3 大深度 + 完整 Java 实现 + 实战选型决策
- **智能优化** — 遗传算法/蚁群算法/模拟退火/粒子群
- **机器学习** — CNN/Transformer/K-Means/决策树

## 三、经典案例

| 场景 | 算法 | 复杂度 | 应用 |
|------|------|--------|------|
| 订单排序 | 快速排序 | O(n log n) | 电商平台 |
| 数据库查询 | 二分查找 | O(log n) | B+Tree 索引 |
| 缓存检索 | 哈希查找 | O(1) | Redis、HashMap |
| 导航路径 | Dijkstra | O((V+E) log V) | 地图软件 |
| 游戏寻路 | A* | — | 游戏角色移动 |
| 人像抠图 | GrabCut | — | 证件照 |
| 数据加密 | RSA | — | 电子商务 / HTTPS |
| 文本搜索 | KMP | O(n+m) | 编辑器查找 |
| 推荐排序 | PageRank | — | 搜索引擎 |
| 任务调度 | 拓扑排序 | O(V+E) | 编译依赖、课程安排 |

## 四、复杂度速查

| 复杂度 | 名称 | 典型算法 | 规模参考 |
|--------|------|---------|---------|
| **O(1)** | 常数 | 哈希查找 | 无限 |
| **O(log n)** | 对数 | 二分查找 | 10⁹ |
| **O(n)** | 线性 | 线性扫描 | 10⁸ |
| **O(n log n)** | 线性对数 | 快排、归并 | 10⁷ |
| **O(n²)** | 平方 | 冒泡、插入排序 | 10⁴ |
| **O(n³)** | 立方 | Floyd、矩阵乘法 | 10³ |
| **O(2ⁿ)** | 指数 | 暴力枚举子集 | ≤ 20 |
| **O(n!)** | 阶乘 | 暴力全排列 | ≤ 10 |

> 详细分析见 [复杂度分析](complexity/) — [时间复杂度](complexity/time-complexity/) · [空间复杂度](complexity/space-complexity/)

## 五、排序算法对比

| 算法 | 平均 | 最好 | 最坏 | 空间 | 稳定 | 适用 |
|------|------|------|------|------|:----:|------|
| 冒泡排序 | O(n²) | O(n) | O(n²) | O(1) | ✓ | 教学 |
| 插入排序 | O(n²) | O(n) | O(n²) | O(1) | ✓ | 小/近有序 |
| 选择排序 | O(n²) | O(n²) | O(n²) | O(1) | ✗ | 教学 |
| 希尔排序 | O(n^1.3) | O(n log n) | O(n²) | O(1) | ✗ | 中等规模 |
| **快速排序** | O(n log n) | O(n log n) | O(n²) | O(log n) | ✗ | 通用首选 |
| **归并排序** | O(n log n) | O(n log n) | O(n log n) | O(n) | ✓ | 稳定排序、外部排序 |
| **堆排序** | O(n log n) | O(n log n) | O(n log n) | O(1) | ✗ | Top-K |
| 计数排序 | O(n+k) | O(n+k) | O(n+k) | O(k) | ✓ | 整数范围小 |
| 桶排序 | O(n+k) | O(n) | O(n²) | O(n+k) | ✓ | 均匀分布 |
| 基数排序 | O(d(n+k)) | O(d(n+k)) | O(d(n+k)) | O(n+k) | ✓ | 整数/字符串 |

## 六、查找算法对比

| 算法 | 复杂度 | 前提 | 适用 |
|------|--------|------|------|
| 线性查找 | O(n) | 无 | 无序小数据集 |
| **二分查找** | O(log n) | 有序数组 | 静态有序数据 |
| **哈希查找** | O(1) 平均 | 哈希表 | 缓存、去重、关联 |
| **BST 查找** | O(log n) 平均 | 二叉搜索树 | 动态有序数据 |
| B+Tree 查找 | O(log n) | B+Tree | 数据库索引 |
| 跳表查找 | O(log n) | 跳表 | Redis ZSet |
