# 算法概述

> 算法是**解决问题的清晰指令序列**——一系列有限步骤的精确描述，将输入转换为输出。

---

## 一、核心特征

| 特征 | 说明 |
|------|------|
| **可行性** | 每一步可分解为基本可执行操作 |
| **确定性** | 每一步明确定义，无歧义 |
| **有穷性** | 在有限步骤内终止 |
| **输入/输出** | 0+ 个输入 → 1+ 个输出 |

> "算法 + 数据结构 = 程序" — 尼克劳斯·维尔特

## 二、分类体系

### 按设计方法
| 类别 | 典型算法 | 特点 |
|------|---------|------|
| **贪心算法** | Dijkstra、就近派单 | 局部最优选择 |
| **动态规划** | 背包问题、编辑距离 | 分解子问题 + 存储中间结果 |
| **分治法** | 快速排序、归并排序、FFT | 分而治之 |
| **随机化** | 蒙特卡罗方法 | 引入随机性 |

### 按应用领域
- **基础算法** — 排序/查找/递归与迭代
- **数据结构相关** — 堆/栈/队列/树/图
- **智能优化** — 遗传算法/蚁群算法/模拟退火
- **机器学习** — CNN/Transformer/K-Means
- **图论** — BFS/DFS/最小生成树/网络流

## 三、经典案例

| 场景 | 算法 | 复杂度 | 应用 |
|------|------|--------|------|
| 订单排序 | 快速排序 | O(n log n) | 电商平台 |
| 数据库查询 | 二分查找 | O(log n) | 索引查找 |
| 缓存检索 | 哈希查找 | O(1) | Redis |
| 导航路径 | Dijkstra | O(n²) | 地图软件 |
| 游戏寻路 | A* | — | 游戏角色 |
| 人像抠图 | GrabCut | — | 证件照 |
| 数据加密 | RSA | — | 电子商务 |
