# 优化算法

> 求解目标函数最优解（最小值/最大值）的迭代方法，是机器学习的核心引擎。

## 为什么需要优化算法？

机器学习的本质是**最小化损失函数**（或最大化似然函数）。损失函数的参数空间通常是高维非凸的，解析解几乎不可能求得，只能靠迭代优化算法逐步逼近最优解。

## 6 大优化算法对比

| 算法 | 核心思想 | 学习率 | 动量 | 自适应 | 适用场景 |
|------|---------|--------|------|--------|---------|
| **SGD** | 沿负梯度方向更新 | 固定/衰减 | ❌ | ❌ | 简单模型、教学 |
| **SGD + Momentum** | 累积历史梯度方向，加速收敛 | 固定 | ✅ | ❌ | 有局部最优的非凸问题 |
| **AdaGrad** | 按历史梯度平方和自适应调整学习率 | 自适应 | ❌ | ✅ | 稀疏特征（NLP） |
| **RMSprop** | 指数加权移动平均替代累积，解决 AdaGrad 学习率过早衰减 | 自适应 | ❌ | ✅ | RNN / 非平稳目标 |
| **Adam** | Momentum + RMSprop 结合，一阶矩 + 二阶矩估计 | 自适应 | ✅ | ✅ | **通用首选**，深度学习默认 |
| **AdamW** | Adam + 解耦权重衰减（正确 L2 正则） | 自适应 | ✅ | ✅ | Transformer / 大模型训练 |

## 选型决策树

```text
任务类型？
├── 教学 / 简单模型 → SGD
├── 稀疏特征（NLP / 推荐） → AdaGrad
├── RNN / 非平稳目标 → RMSprop
├── 通用深度学习 → Adam
└── Transformer / 大模型 → AdamW（推荐）
```

## 关键概念

- **学习率（Learning Rate）**：控制每步更新幅度。太大→震荡发散，太小→收敛慢/陷入局部最优。常用策略：warmup + cosine decay
- **动量（Momentum）**：累积历史梯度方向，平滑更新轨迹，加速穿过平坦区域。典型值 β=0.9
- **凸 vs 非凸**：凸函数有唯一全局最优（如线性回归的 MSE）；非凸函数有多个局部最优和鞍点（如神经网络）
- **鞍点问题**：高维空间中鞍点远多于局部最优，SGD 容易卡住；Momentum / Adam 可借助惯性冲出

## 常见陷阱

| 陷阱 | 症状 | 修复 |
|------|------|------|
| 学习率过大 | loss 震荡甚至 NaN | 降低学习率 / 加 gradient clipping |
| 学习率过小 | loss 下降极慢 | 增大学习率 / 加 warmup |
| 梯度爆炸 | loss 突然飙升 | gradient clipping（max_norm=1.0） |
| 梯度消失 | loss 几乎不下降 | 换激活函数（ReLU）/ BatchNorm / 残差连接 |
