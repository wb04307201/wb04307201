# 搜索算法

> ⬅️ [返回 算法概述](../README.md)

> **一句话定位**：搜索算法 = **在状态空间中寻找目标节点**的系统性方法。从盲目枚举（BFS/DFS）到启发式（A*）、从精确求解（分支限界）到近似（局部搜索），覆盖组合优化的全谱。

---

## 🎯 学习目标

完成本节后，你能够：

- **盲目搜索**：区分 BFS / DFS / 迭代加深的适用场景与复杂度
- **启发式搜索**：说出 A* 的一致可采纳条件，理解 f = g + h 的物理含义
- **精确求解**：掌握分支限界（B&B）的"分支 + 剪枝"原理
- **二分搜索**：能处理有序数组中的 6 大变体（找第一个/最后一个/左边界等）

## 🧭 选型决策树

```text
已知解路径长度？
├── 是 + 最短 → BFS（广度优先）
├── 否 + 任意解 → DFS（深度优先）
└── 有启发式信息 → A*（最优启发式）

精确解 or 近似解？
├── 精确 + 小规模 → 回溯 / 分支限界
├── 精确 + 大规模 → 启发式 + 剪枝（A* / IDA*）
└── 近似 → 局部搜索 / 模拟退火 / GA

有序数组查找？
└── 二分搜索（左闭右开、左闭右闭 6 大变体）
```
