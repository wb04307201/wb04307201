# 计算机网络基础

> 计算机网络是将地理位置不同、具有独立功能的多台计算机及其外部设备，通过通信线路和设备连接起来，在网络操作系统、协议和管理软件的协调下，实现**资源共享**和**信息传递**的计算机系统。

---
## 引言：反直觉代码
计算机网络基础 的关键不是语法——是**看起来对**的代码背后那些'踩坑点'。

本篇用 3 个反直觉片段切入，把面试/生产中常被问起、但一深入就漏馅的点摆出来。

---

## 一、网络组成

| 子网 | 职责 | 核心组件 |
|------|------|---------|
| **资源子网**（终端系统） | 提供共享资源 | 服务器、PC、打印机、数据库 |
| **通信子网**（传输系统） | 负责数据转发 | 路由器、交换机、通信线路 |

## 二、核心功能

- **资源共享** — 硬件（打印机/存储）、软件（应用/数据库）、信息（文件/数据）
- **数据通信** — 计算机之间、终端之间的数据传输（最基本功能）
- **分布式处理** — 大任务拆分到多台计算机并行执行
- **负载均衡** — 任务过重时将部分工作转移到其他节点

## 三、分类方式

| 分类维度 | 类型 | 覆盖范围 |
|---------|------|---------|
| **地理范围** | LAN（局域网） | 办公室/建筑物内 |
| | MAN（城域网） | 城市级别 |
| | WAN（广域网） | 国家/全球 |
| **拓扑结构** | 星型、总线型、树型、网型、环型 | — |
| **传输介质** | 有线网络（双绞线/光纤） | 物理线路传输 |
| | 无线网络（WiFi/蓝牙/5G） | 无线电波传输 |

## 四、数据交换方式

| 方式 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| **电路交换** | 建立专用通路，全程独占 | 实时性高、延迟稳定 | 信道浪费、并发差 |
| **报文交换** | 整报文存储转发 | 信道利用率高 | 大报文可能接收不完整 |
| **分组交换** | 报文分片后分组传输 | 灵活高效、并发好 | 数据分散、需重组 |

> **现代互联网以分组交换为主**（TCP/IP 协议族的基础）。

## 五、协议与体系结构

### 网络协议三要素
- **语法** — 数据格式、编码规则
- **语义** — 控制信息的含义（动作/响应）
- **时序** — 事件执行顺序

### 两大参考模型
- **[OSI 七层模型](osi-model/)** — 理论标准（物理层→数据链路层→网络层→传输层→会话层→表示层→应用层）
- **[TCP/IP 四层模型](tcp-ip-model/)** — 实际标准（链路层→网络层→传输层→应用层）

## 六、应用领域

| 领域 | 典型应用 |
|------|---------|
| 信息通信 | 邮件、即时通讯、视频会议 |
| 远程访问 | VPN、远程桌面 |
| 分布式计算 | 云计算、大数据处理 |
| 电子商务 | 在线支付、网上银行 |
| 在线服务 | 流媒体、网络游戏 |
