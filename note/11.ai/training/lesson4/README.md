# 第 4 课：Java MCP 服务开发

> 使用 Java 开发 MCP Server 并发布到 Maven 中央仓库，让你的工具服务被所有 MCP 客户端调用。

---

## 学习目标

学完本课后，你将能够：

- 理解 MCP Server 的项目架构和核心概念
- 使用 Spring Boot 创建一个 MCP Server
- 实现天气查询等工具服务（Tools）
- 配置 MCP Server 的 SSE 传输模式
- 将 MCP Server 发布到 Maven 中央仓库
- 使用 JBang 快速测试 MCP Server

## 前置条件

- 前置课程：[第 3 课：Spring AI Agent 搭建](../lesson3/README.md)
- 环境准备：JDK 17+、Maven 3.8+、GPG 密钥（发布用）
- Maven Central 账号（Sonatype JIRA）
