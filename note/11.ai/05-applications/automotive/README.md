## 🚗 智能驾驶：端到端大模型时代

### 三代技术演进

```text
L1-L2（2015-2020）: 规则 + 模块化
  感知 → 定位 → 预测 → 规划 → 控制（5 个独立模块）
  代表: Mobileye EyeQ 系列 / Bosch

L2+（2020-2023）: BEV + Transformer
  Bird's Eye View 统一感知 + Transformer 时序融合
  代表: Tesla FSD V11 / 华为 ADS 2.0 / 小鹏 XNGP

L3-L4（2023-）: 端到端大模型
  传感器 → 单一大模型 → 控制信号（一段式）
  代表: Tesla FSD V12 / Waymo 6th Gen / 蔚来 NWM
```

### Tesla FSD V12 范式革命

**核心变化**：从 30 万行 C++ 代码 → 神经网络单一模型。

```text
旧架构（V11）：
  摄像头 → 感知网络 → 规则引擎 → 规划器 → 控制
  30 万行规则代码维护成本极高

新架构（V12）：
  摄像头 → 单一神经网络 → 控制信号
  数据驱动 + 影子模式（shadow mode）
```

**关键技术**：
- **BEV + Transformer**：鸟瞰图统一空间
- **占用网络（Occupancy Network）**：3D 体素预测替代物体检测
- **隐式规划（Implicit Planning）**：端到端输出轨迹
- **影子模式**：人工驾驶时模型同步预测，验证一致性

## 🎙️ 智能座舱：多模态 Agent

### 核心能力矩阵

| 能力 | 技术栈 | 代表场景 |
|------|--------|---------|
| **语音交互** | ASR + LLM + TTS | "嗨，NOMI，导航到公司" |
| **多模态理解** | VLM（视觉语言模型） | 摄像头识别手势 / 表情 |

## 🛠️ 制造：智能工厂

### 视觉质检（AI 替代人工）

**传统**：30 人目检 / 班次，漏检率 3-5%
**AI 视觉**：99.5% 准确率，漏检率 < 0.5%

**主流方案**：
- 凌云光 / 海康威视：通用视觉平台
- 创新奇智 / 旷视：垂直行业方案
- 自研：基于 YOLOv8 / RT-DETR 微调

### 预测性维护

```text
传感器数据 → 时序模型（LSTM / Transformer）
   ↓
剩余寿命（RUL）预测
   ↓
提前 7 天预警，安排检修
```

ROI：减少 70% 非计划停机，节省 30% 维护成本。

## 💼 销售售后：AI Agent 重构服务

### 数字销售顾问

```text
用户进店 → AI 摄像头识别（VIP / 老客户）
   ↓
推送个性化话术 + 历史偏好
   ↓
试驾时实时生成定制化演示
   ↓
离店后 AI 跟进 + 智能邀约
```

- **故障诊断**：OBD 数据 + LLM 推理，识别 90% 常见故障
- **预约调度**：智能排产 + 用户偏好匹配
- **配件预测**：基于区域销售预测，提前 30 天备货
