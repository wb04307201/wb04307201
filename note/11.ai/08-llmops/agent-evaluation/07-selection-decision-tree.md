# Agent 评测选型决策 · 按阶段 / 团队 / 预算

> **一句话**：没有"统一评测方案"——按 **阶段**（MVP / 增长 / 成熟）+ **预算** + **团队**选对应方案。新团队从 MVP 开始，老团队补自动化。

← [返回: Agent Evaluation 总目录](../README.md)

---

## 1. 5 分钟决策树

```text
Q1：项目阶段？
├─ MVP（刚上线）→ 自动化指标 + 黄金集 50 条 + LLM-judge
├─ 增长（用户 1k-100k）→ + A/B Test + 漂移检测
└─ 成熟（用户 > 100k）→ + 用户模拟 + 多模型投票 + 自动回滚

Q2：团队规模？
├─ 1-3 人 → 只做关键流程（自动化 + 黄金集）
├─ 5-10 人 → + CI 集成 + LLM-judge
└─ > 10 人 → + 用户模拟 + 漂移告警 + 自动回滚

Q3：预算？
├─ < $1k/月 → 黄金集 + 自动化（低成本）
├─ $1k-10k/月 → + LLM-judge 多模型投票
└─ > $10k/月 → + 用户模拟（GPU）+ 漂移监控系统

Q4：任务类型？
├─ 结构化（订机票 / 退款）→ 自动化为主
├─ 开放式（对话 / 写作）→ LLM-judge 为主
└─ 多步工具 → Trajectory 分析 + 工具准确率

Q5：是否上线生产？
├─ 否（demo）→ 离线评测够
└─ 是 → A/B Test + 在线监控 + 自动回滚
```

## 2. 3 阶段选型矩阵

| 阶段 | 黄金集 | 自动化 | LLM-judge | A/B Test | 用户模拟 | 漂移检测 |
|------|--------|--------|-----------|----------|---------|----------|
| **MVP** | 50 条 | ✅ | ✅ 单模型 | ❌ | ❌ | ❌ |
| **增长** | 200 条 | ✅ | ✅ 多模型 | ✅ 5% | ⚠️ 轻量 | ✅ 告警 |
| **成熟** | 500+ 条 | ✅ | ✅ 多模型投票 | ✅ 多梯度 | ✅ 1000+ | ✅ 自动回滚 |

---

## 3. 任务类型选型

| 任务类型 | 主方法 | 辅助方法 | 阈值建议 |
|---------|--------|---------|----------|
| **结构化任务** | 自动化指标 | 黄金集 | ≥ 0.95 |
| **对话 / 客服** | LLM-judge | 用户满意度 | ≥ 0.85 |
| **代码助手** | 自动化（编译/测试） | LLM-judge | ≥ 0.80 |
| **研究 Agent** | LLM-judge + 专家评审 | 用户模拟 | ≥ 0.75 |
| **多步工具** | 工具准确率 + 步骤效率 | trajectory 分析 | ≥ 0.80 |

---

## 4. 实施 Checklist

### 4.1 MVP 阶段（Day 1）
- [ ] 标注 50 条黄金集（业务专家手工）
- [ ] 写自动化指标（Schema + 字段匹配）
- [ ] 接 GPT-4 judge（先单一，后续多模型）
- [ ] 发版前跑黄金集 + judge

### 4.2 增长阶段（Week 2-4）
- [ ] CI 集成（PR 触发 eval）
- [ ] 多模型 judge（GPT-4 + Claude + Gemini）
- [ ] A/B Test 灰度 5% 流量
- [ ] 在线监控（完成率 + 延迟 + 成本）

### 4.3 成熟阶段（Month 2+）
- [ ] 漂移检测 + 自动告警
- [ ] 每周补充失败用例 → 黄金集增长
- [ ] 用户模拟（1000+ 并发）
- [ ] 自动回滚机制
- [ ] 季度 review 黄金集

---

## 5. 反模式速查（5 大错）

| 反模式 | 症状 | 修复 |
|--------|------|------|
| 跳到成熟阶段 | 小团队直接用户模拟 | 阶段化，从小到大 |
| 无区分度阈值 | 阈值 100% 或太低 | 阈值 80-85%，按业务调整 |
| 无失败用例收集 | 用户投诉不进黄金集 | 周更黄金集 |
| 无 CI 集成 | 评测靠人工跑 | 自动化 CI |
| 无漂移告警 | 模型退化几周才发现 | 每日漂移检测 |

---

## 6. 一句话总结

> **Agent 评测选型 = 阶段化（MVP 50 条 → 增长 200 条 → 成熟 500 条）+ 任务类型（结构化 / 对话 / 多步）+ 预算（<$1k / $1k-10k / >$10k）。没有银弹，按需组合 5 种方法。**
