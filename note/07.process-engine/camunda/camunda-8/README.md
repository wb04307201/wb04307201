# Camunda 8

Camunda Platform 8于2022年4月12日发布，其核心引擎就是Zeebe，Camunda公司还有传统的开源工作流引擎Camunda Platform 7，是唯一一家同时开源2种工作流引擎的公司。

在国外自从Camunda BPM 7.0.0推出后，很多客户陆续从Activit迁移到Camunda BPM。
